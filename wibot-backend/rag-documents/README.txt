WIBOT RAG Documents
===================

Structure des dossiers :

/procedures/    - Procedures techniques (reset password, panne internet, etc.)
/clients/       - Fiches clients (EHPAD, cliniques, contacts)
/tickets/       - Historique tickets resolus
/documentation/ - Documentation technique WIDIP

Formats supportes :
- Texte : .md, .txt, .json, .csv, .html
- PDF : .pdf (extraction automatique)
- Word : .docx (extraction automatique)
- Excel : .xlsx, .xls (extraction automatique)

Les fichiers sont automatiquement classes par categorie
selon leur dossier parent.
